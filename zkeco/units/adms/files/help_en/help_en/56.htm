The visitor system is highly integrated with the access control system and elevator control system. Door access rights must be granted to visitors. ?Note: Complete the following settings before using the USB camera-based capture function: Right-click Capture in the capture frame and choose Global Settings from the shortcut menu. On the Storage tab page, click Local Storage Settings by Site to add a website address. Add 127.0.0.1 if the computer is a server, and add the IP address of the server such as 192.168.8.12 if the computer is a client. Set the website to be allowed on the Camera and Mic and Playback tab pages. The settings need to be completed only once as long as history records of the browser are not deleted. If the camera does not capture any pictures, check whether the settings are correct and whether the camera indicator is on, close and then open the browser again.

?Note: The camera can be loaded and invoked only by one page at a time.
