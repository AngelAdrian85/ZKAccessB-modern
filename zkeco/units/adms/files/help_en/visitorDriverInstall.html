Visitor Device Driver Installation


Please install visitor device driver if you want to enroll visitor through the system.


Click here to download visitor device driver!


Unzip after download Visitor Driver.rar. The visitordriver folder copy all the files to windows\system32 (windows\system in win98)


Copy after found in the system directory the register.bat, double click to run.


Note:


1. Make sure you are using the computer C drive visitors otherwise not be able to use the device registration;


2. Run register.bat file as administrator under win7.
