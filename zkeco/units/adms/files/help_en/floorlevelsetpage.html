Elevator Levels


Elevator levels means in a specific time period, which floor or floor combination can be opened through verification. However, the personnel combination that can open these floors via verification shall be set in personnel elevtor levels settings. Please refer to 7.5 Personnel elevator levels settings.


Add Elevator levels:


1. Click [Elevator Control] - [Elevator levels] - [Add] to enter Add elevator levels edit interface;


2.Set parameters: elevator level name (no repetition), elevator control time zones, floor combination;


3.Click [OK] to complete setting and quit, and added elevator levels will appear in the list.


    Note:


    (1)Select the floors in the elevator levels as multi-choice, so you can select different floors in different control panels;


    (2)Two levels with the same time zone and floor combination are not allowed in the system.
